## Системные требования

| Компонент | Минимум | Рекомендуется |
|---|---|---|
| ОС | Windows 10 (1903+) 64-бит | Windows 11 |
| Процессор | Intel Core i3 / AMD Ryzen 3 (2015+) | Core i5 / Ryzen 5 (2020+) |
| ОЗУ | 8 ГБ | 16 ГБ |
| Место на диске | 500 МБ | 1 ГБ |
| Интернет | Требуется | Стабильный ≥ 10 Мбит/с |
| Микрофон | Любой встроенный | Внешний USB/3.5мм |

## Установка (после релиза)

### 1. Скачайте установщик

Перейдите на страницу [Releases](../../releases) и скачайте `NOVA-Setup-x64.exe`.

### 2. Запустите установщик

Дважды кликните по файлу. Если Windows SmartScreen показывает предупреждение — нажмите **«Подробнее» → «Всё равно запустить»**.

Следуйте инструкциям установщика:
- Выберите папку установки (по умолчанию `C:\Program Files\NOVA Assistant`)
- Выберите, создавать ли ярлык на рабочем столе
- Нажмите **Установить**

### 3. Первый запуск

При первом запуске NOVA спросит:

**Доступ к микрофону** — обязательно разрешите, иначе голосовые команды не будут работать.

**API-ключ** — введите один из поддерживаемых ключей:

| Провайдер | Где получить | Рекомендация |
|---|---|---|
| OpenAI | [platform.openai.com](https://platform.openai.com/api-keys) | GPT-4o — лучшее качество |
| Anthropic | [console.anthropic.com](https://console.anthropic.com/) | Claude — быстрый и качественный |
| Groq | [console.groq.com](https://console.groq.com/) | Бесплатный, очень быстрый |

### 4. Проверка

После настройки скажите:

> **«Нова, привет»**

Если ассистент ответил — установка прошла успешно. Вы увидите NovaIsland в правом углу экрана.

## Настройка микрофона в Windows

Если NOVA не слышит вас, проверьте настройки:

1. **Пуск → Параметры → Конфиденциальность → Микрофон**
2. Убедитесь, что переключатель **«Разрешить приложениям доступ к микрофону»** включён
3. Прокрутите вниз и включите доступ для **NOVA Assistant**

Также проверьте:
- **Пуск → Параметры → Система → Звук → Входное устройство** — должен быть выбран ваш микрофон
- Протестируйте микрофон кнопкой **«Проверить микрофон»**

## Обновление

NOVA проверяет обновления автоматически при каждом запуске. Когда выйдет новая версия, вы увидите уведомление в трее.

Обновить вручную: **Настройки → О программе → Проверить обновления**

---

## Удаление

**Пуск → Параметры → Приложения → найдите «NOVA Assistant» → Удалить**

Все настройки и данные хранятся в `%APPDATA%\NOVA Assistant`. Удалите эту папку вручную, если хотите полностью очистить данные.

## Проблемы при установке

**«Windows защитила ваш компьютер»**  
Нажмите «Подробнее» → «Всё равно запустить». Это стандартное предупреждение для новых приложений без EV-сертификата.

**Ошибка «Требуется Visual C++ Redistributable»**  
Скачайте и установите [Microsoft Visual C++ 2019+](https://aka.ms/vs/17/release/vc_redist.x64.exe).

**Ошибка «Требуется .NET Runtime»**  
Скачайте [.NET Desktop Runtime](https://dotnet.microsoft.com/download/dotnet/8.0).

**Установщик не запускается**  
Убедитесь, что у вас Windows 10 версии 1903 и выше. Проверьте: **Пуск → winver**.
