## Что такое NOVA?

NOVA — голосовой ассистент для Windows, который управляет компьютером по голосу: открывает приложения, регулирует звук, управляет файлами, ищет в браузере и отвечает на вопросы через ChatGPT / Claude.

В отличие от облачных ассистентов, NOVA работает как настольное приложение прямо на вашем ПК.

## Системные требования

| Компонент | Минимум |
|---|---|
| ОС | Windows 10 / 11 (64-бит) |
| Процессор | Intel Core i3 или AMD Ryzen 3 (2015+) |
| ОЗУ | 8 ГБ и более |
| Интернет | Требуется для активации, STT/TTS и ИИ-моделей |
| Микрофон | Любой, встроенный или внешний |

## Установка

> **Статус:** Публичный установщик появится после релиза. Следите за обновлениями.

После выхода установщика:

1. Скачайте файл `NOVA-Setup-x64.exe` со страницы [Releases](../../releases).
2. Запустите установщик и следуйте инструкциям.
3. При первом запуске введите API-ключ (OpenAI, Anthropic или Groq) в настройках.
4. Разрешите доступ к микрофону.
5. Скажите «Нова» — ассистент готов к работе.

## Как использовать

### Wake word

По умолчанию ассистент активируется фразой **«Нова»**:

```
Нова, открой браузер
Нова, сделай скриншот
Нова, какая погода?
```

### Удержание кнопки

Нажмите и удерживайте кнопку микрофона в интерфейсе, чтобы отдать команду без wake word.

### NovaIsland

Небольшая плавающая панель на экране показывает статус ассистента:
- **Жду** — ассистент готов, ожидает wake word
- **Слушаю** — распознаёт вашу речь
- **Отвечаю** — генерирует или озвучивает ответ
- **Выполняю** — исполняет системную команду
- **Сплю** — ассистент в режиме ожидания

## Голосовые команды

Полный список — в файле [COMMANDS.md](./COMMANDS.md). Ниже — самые популярные.

### Пользовательские команды (конструктор)

В разделе **«Команды»** можно собирать собственные сценарии в node-graph:

- узлы: `Триггер`, `Действие`, `Условие`, `Вывод`, `Ответ`, `Конец`;
- поддерживаются последовательные действия, ветвления и повторные подтверждения команды;
- в узле **«Ответ»** ассистент задаёт вопрос и выбирает ветку `Да/Нет` по вашей фразе.

Пример:  
`Нова, активируй протокол я дома` → открыть приложения → настроить окна → спросить подтверждение следующего шага → завершить сценарий.

## FAQ — Часто задаваемые вопросы

**Нова не слышит меня. Что делать?**  
Проверьте, что микрофон выбран как устройство ввода по умолчанию в Windows → Звук → Запись. Убедитесь, что приложение имеет разрешение на доступ к микрофону (Параметры → Конфиденциальность → Микрофон).

**Нужен ли постоянный интернет?**  
Интернет требуется для: распознавания речи (STT), синтеза речи (TTS) и ответов ИИ. Без интернета доступны только офлайн-системные команды.

**Какие API-ключи поддерживаются?**  
OpenAI (GPT-4o, GPT-3.5), Anthropic (Claude), Groq. Ключи хранятся локально и не передаются третьим сторонам.

**Ассистент слишком медленно реагирует.**  
Попробуйте переключиться на более быструю модель (например, Groq Llama) в настройках → Провайдер.

**Как изменить wake word?**  
Настройки → Голос → Wake word. Можно задать своё слово активации.

**NovaIsland мешает. Как скрыть?**  
Скажите «Нова, скрой остров» или откройте настройки → Интерфейс.

**Работает ли NOVA на Windows 10?**  
Да, поддерживаются Windows 10 (версия 1903+) и Windows 11.

**Где хранятся мои данные?**  
Все данные хранятся локально на вашем ПК. NOVA не собирает и не передаёт данные без вашего ведома.
